# ShopLuxe33 — première version

## Ouvrir le site
Double-clique sur `index.html`.

## Modifier les produits
Ouvre `index.html` avec le Bloc-notes ou Visual Studio Code.
Recherche `product-card`, puis change :
- le nom ;
- le prix ;
- la catégorie ;
- la photo.

## Ajouter une vraie photo
1. Place la photo dans le dossier `assets`.
2. Remplace :
   `<div class="product-image placeholder-image"><span>Photo produit 1</span></div>`
   par :
   `<div class="product-image"><img src="assets/nom-photo.jpg" alt="Nom du produit"></div>`
3. Ajoute dans `style.css` :
   `.product-image img { width: 100%; height: 100%; object-fit: cover; }`
